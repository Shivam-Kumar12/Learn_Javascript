About Functions

A function is a block of code designed to perform a particular task. Functions are executed when they are called (invoked). This allows for code reuse and better organization.

Syntax

    
// Function Declaration
function functionName(parameters) {
  // Code to be executed
}

// Function Call
functionName(arguments);

Example 1: Function Declaration and Invocation

Here is an example of a function that adds two numbers:

    
function add(a, b) {
  return a + b;
}

let sum = add(2, 3);
console.log(`Sum is: ${sum}`); // Output: Sum is: 5

Example 2: Arrow Function

Arrow functions provide a shorter syntax for writing functions:

    
const subtract = (a, b) => {
  return a - b;
};

let difference = subtract(10, 4);
console.log(`Difference is: ${difference}`); // Output: Difference is: 6
