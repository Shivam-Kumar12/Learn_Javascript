About setInterval

The setInterval method in JavaScript is used to repeatedly call a function or execute a code snippet, with a fixed time delay between each call. It continues to execute the code until clearInterval is called or the window is closed.

Syntax

    
setInterval(function, delay);

Parameters:

  • function: The function to be executed.
  • delay: The time, in milliseconds (thousandths of a second), the timer should wait before the specified function is executed.

About clearInterval

The clearInterval method in JavaScript is used to stop the execution of a function that was set up using setInterval. You need to pass the identifier returned by setInterval to clearInterval.

Syntax

    
clearInterval(intervalID);

Parameters:

  • intervalID: The identifier returned by setInterval that specifies which interval to clear.

About setTimeout

The setTimeout method in JavaScript is used to call a function or execute a code snippet after a specified delay. It only executes the code once after the delay has elapsed.

Syntax

    
setTimeout(function, delay);

Parameters:

  • function: The function to be executed.
  • delay: The time, in milliseconds (thousandths of a second), the timer should wait before the specified function is executed.

About clearTimeout

The clearTimeout method in JavaScript is used to cancel the execution of a function that was set up using setTimeout. You need to pass the identifier returned by setTimeout to clearTimeout.

Syntax

    
clearTimeout(timeoutID);

Parameters:

  • timeoutID: The identifier returned by setTimeout that specifies which timeout to clear.
