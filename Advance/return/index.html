About the `return` Keyword

The return keyword is used to return a value from a function. When a return statement is encountered, the function stops executing and the specified value is returned to the caller.

Syntax

    
function functionName(parameters) {
  // code to be executed
  return value;
}

Example: Function with `return`

Here is an example of a function that multiplies three numbers and returns the result:

    
function multiply(num1, num2, num3) {
  return num1 * num2 * num3;
}

let result = multiply(2, 3, 4);
console.log(`The result is: ${result}`); // Output: The result is: 24
