About the Do While Loop

The do...while loop is similar to the while loop, but the condition is evaluated after the execution of the loop's body. This means that the loop's body will always execute at least once, even if the condition is false initially.

Syntax

    
do {
  // Code to execute in each iteration
} while (condition);

Example

Here is an example of a do...while loop that logs the numbers from 0 to 4:

    
let count = 0;
do {
  console.log(`Count is: ${count}`);
  count++;
} while (count < 5);

Example Explained

In the example above:

  • The variable count is initialized to 0.
  • The code inside the do block executes, logging the current value of count and then incrementing count by 1.
  • After the code block, the condition count < 5 is evaluated.
  • If the condition is true, the loop repeats; if false, the loop terminates.

Preventing Infinite Loops

Ensure that the condition will eventually become false to avoid creating an infinite loop. For example:

    
let count = 0;
do {
  console.log("This will run forever!");
} while (true);

Exiting the Loop Prematurely

You can use the break statement to exit a do...while loop before the condition becomes false:

    
let count = 0;
do {
  console.log(`Count is: ${count}`);
  count++;
  if (count === 3) {
    break; // Exit the loop when count is 3
  }
} while (count < 5);

Skipping Iterations

You can use the continue statement to skip the current iteration and move to the next one:

    
let count = 0;
do {
  count++;
  if (count === 3) {
    continue; // Skip the rest of the code in this iteration when count is 3
  }
  console.log(`Count is: ${count}`);
} while (count < 5);
