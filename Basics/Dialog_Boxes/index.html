Dialog Boxes

There are three types of Dialog Boxes

    The alert() method displays a simple message in a dialog box with an OK button. It is often used to convey information to the user.

  1. alert("Your message goes here.")

    2. The confirm() method displays a dialog box with a specified message, along with OK and Cancel buttons. It returns true if the user clicks OK, and false if the user clicks Cancel. It is used to confirm an action from the user.

  2. confirm("Your message goes here.")

    3. The prompt() method displays a dialog box that prompts the user to input some text. It returns the text entered by the user if OK is clicked, or null if Cancel is clicked. It is used to collect input from the user

  3. prompt("Your message goes here.")
