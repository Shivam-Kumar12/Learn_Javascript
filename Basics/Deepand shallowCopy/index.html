Shallow and Deep Copies in JavaScript

Shallow Copy

A shallow copy means that only the first level of the object or array is copied. The nested objects or arrays within the copied object/array are still referenced. 


// Using Object.assign() for a shallow copy
const originalObject = { a: 1, b: { c: 2 } };
const shallowCopy = Object.assign({}, originalObject);
console.log(shallowCopy); // Output: { a: 1, b: { c: 2 } }

// Using spread operator for a shallow copy
const originalArray = [1, [2, 3]];
const shallowArrayCopy = [...originalArray];
console.log(shallowArrayCopy); // Output: [1, [2, 3]]

Deep Copy

A deep copy means that all levels of the object or array are copied, and the nested objects or arrays within the copied object/array are not referenced. Instead, new copies of all nested objects or arrays are created. 


// Using JSON methods for a deep copy
const originalObject2 = { a: 1, b: { c: 2 } };
const deepCopy = JSON.parse(JSON.stringify(originalObject2));
console.log(deepCopy); // Output: { a: 1, b: { c: 2 } }

Examples

Shallow Copy Example:


const shallowCopyExample = { x: 1, y: { z: 2 } };
const shallowCopyOfExample = { ...shallowCopyExample };

shallowCopyOfExample.y.z = 5;
console.log(shallowCopyExample.y.z); // Output: 5 (The original object's nested property is affected)

Deep Copy Example:


const deepCopyExample = { x: 1, y: { z: 2 } };
const deepCopyOfExample = JSON.parse(JSON.stringify(deepCopyExample));

deepCopyOfExample.y.z = 5;
console.log(deepCopyExample.y.z); // Output: 2 (The original object's nested property is not affected)
