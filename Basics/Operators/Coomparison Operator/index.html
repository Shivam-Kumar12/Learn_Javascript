JavaScript Comparison Operators

Comparison operators are used to compare two values. They return a boolean value, either true or false. Here are some common comparison operators in JavaScript:

Operators

  • == : Equal to
  • === : Strict equal to (checks both value and type)
  • != : Not equal to
  • !== : Strict not equal to (checks both value and type)
  • > : Greater than
  • >= : Greater than or equal to
  • < : Less than
  • <= : Less than or equal to
