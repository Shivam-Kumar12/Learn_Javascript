JavaScript Assignment Operators

Assignment operators are used to assign values to variables. Here are some common assignment operators in JavaScript:

Operators

  • = : Simple assignment
  • += : Addition assignment
  • -= : Subtraction assignment
  • *= : Multiplication assignment
  • /= : Division assignment
  • %= : Modulus assignment
  • **= : Exponentiation assignment
