JavaScript Operators

JavaScript has a variety of operators that can be used to perform different operations on variables and values. Here is a list of some common operators:

Arithmetic Operators

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus (Remainder)
  • ++ : Increment
  • -- : Decrement

Comparison Operators

  • == : Equal to
  • === : Strict equal to
  • != : Not equal to
  • !== : Strict not equal to
  • > : Greater than
  • >= : Greater than or equal to
  • < : Less than
  • <= : Less than or equal to

Logical Operators

  • && : Logical AND
  • || : Logical OR
  • ! : Logical NOT

Assignment Operators

  • = : Assignment
  • += : Addition assignment
  • -= : Subtraction assignment
  • *= : Multiplication assignment
  • /= : Division assignment
  • %= : Modulus assignment

Bitwise Operators

  • & : AND
  • | : OR
  • ^ : XOR
  • ~ : NOT
  • << : Left shift
  • >> : Right shift
  • >>> : Unsigned right shift

Other Operators

  • typeof : Returns the type of a variable
  • instanceof : Checks if an object is an instance of a specific class
  • ? : Ternary operator (conditional)
  • ... : Spread operator
