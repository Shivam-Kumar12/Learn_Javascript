JavaScript Arithmetic Operators

Arithmetic operators are used to perform mathematical operations on numbers. Here are some common arithmetic operators in JavaScript:

Operators

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus (Remainder)
  • ++ : Increment
  • -- : Decrement
