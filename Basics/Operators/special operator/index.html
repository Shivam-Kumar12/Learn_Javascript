JavaScript Special Operators

JavaScript includes several special operators that provide additional functionality and convenience. Here are some notable ones:

Operators

  • ?? : Nullish Coalescing Operator
    Returns the right-hand operand when the left-hand operand is null or undefined; otherwise, it returns the left-hand operand.
  • ?. : Optional Chaining
    Allows accessing deeply nested properties without explicitly checking if each reference in the chain is null or undefined.
  • ?: : Conditional (Ternary) Operator
    A shorthand for the if-else statement, returns one of two values depending on a condition.
  • typeof : Type Operator
    Returns a string indicating the type of the unevaluated operand.
  • instanceof : Instance Operator
    Checks if an object is an instance of a particular class or constructor.
  • in : Property Check Operator
    Checks if a specified property exists in an object or its prototype chain.
  • delete : Deletion Operator
    Removes a property from an object.
